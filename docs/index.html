Gohari Laravel Repository Pattern

Build clean repositories for Laravel models with one Artisan command. Generate repository classes, interfaces, and reusable Eloquent repository methods quickly.

Install Package View on GitHub

Installation

composer require gohari/repository-pattern

Quick Start

php artisan repository:make User --model=User

This command creates a repository and interface for your Laravel model. 

app/Repositories/UserRepository.php
app/Repositories/UserRepositoryInterface.php

Features

Repository Generator

Create clean repository classes with one Artisan command.

Interface Support

Generate matching interfaces for better dependency injection and clean architecture.

Reusable Base Methods

Use common Eloquent methods like getAll, paginate, findById, create, update, and delete.

Example Usage

use App\Repositories\UserRepositoryInterface;

class UserService
{
    public function __construct(
        private readonly UserRepositoryInterface $users
    ) {}

    public function activeUsers()
    {
        return $this->users
            ->query()
            ->where('is_active', true)
            ->get();
    }
}
