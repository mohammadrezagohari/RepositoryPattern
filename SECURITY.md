# Security Policy

## Supported Versions

Security fixes are provided for the latest stable release of `gohari/repository-pattern`.

| Version | Supported |
| --- | --- |
| `v1.1.x` | Yes |
| `< v1.1.0` | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Report vulnerabilities by email:

```text
developer.gohari@gmail.com
```

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Affected package version
- Any relevant proof of concept or logs

You should receive an initial response within 72 hours. If the vulnerability is confirmed, a fix will be prepared and released as soon as reasonably possible.
